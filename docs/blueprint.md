# **App Name**: RiseForge

## Core Features:

- User Account & Onboarding: Secure signup/login with email/password, including a personalized and interactive onboarding experience that gathers user interests and goals.
- Gamified Learning World Map: Interactive 'World Map' progression system featuring themed learning paths, each containing progressive structured lessons with engaging introductions, explanations, real-life examples, interactive questions, and mini-challenges.
- Idea Lab (AI-Powered): A core tool where users can define, structure, and refine business ideas. The system will provide AI-powered improvement suggestions, encouraging feedback, and smart prompts to foster innovation.
- Challenge Arena: Weekly entrepreneurial missions presented as 'quests' where users apply their learning. Completing challenges awards XP, badges, and streak bonuses, simulating real-world problem-solving.
- Gamification Engine & Leaderboard: Comprehensive XP, level, badge, and streak system with real-time animated counters. Track and display user progress on a global and national leaderboard to encourage engagement and healthy competition.
- Skill Tree Progression: A visual skill tree that expands as users complete lessons and challenges, tracking the development of entrepreneurial skills like Critical Thinking, Opportunity Spotting, and Leadership.
- Personalized Dashboard: A 'Mission Control' dashboard featuring a modern grid layout that displays daily motivation, current XP, level, active course, today's challenge, and ongoing idea projects, with animated progress indicators.

## Style Guidelines:

- Default Theme (Dark): A deep navy background (#241F2E) providing a sophisticated base for a strategic growth game. Primary elements will feature an electric violet (#8F63E9) to signify innovation and aspiration. A vibrant analogous blue (#8CE1F2) serves as an accent for calls to action and highlights, drawing on the 'electric blue' suggested for dark mode.
- Light Theme: A soft white background for clarity, with calm blue accents derived from the dark theme's primary hue, ensuring smooth transitions and visual consistency across themes.
- Headline font: 'Poppins' (geometric sans-serif) for its precise, contemporary, and avant-garde aesthetic, perfectly suited for impactful headlines. Body font: 'Inter' (grotesque-style sans-serif) to provide a modern, objective, and highly legible text experience for detailed content.
- Sleek, minimalist line icons with abstract or geometric forms, reflecting a premium startup aesthetic and avoiding any childish or cartoonish imagery.
- Modern grid layouts utilizing 'glass card' effects and a strong visual hierarchy, ensuring a clean and clutter-free experience across all responsive, mobile-friendly designs. Visual elements will feature floating glowing shapes and animated gradient backgrounds.
- Extensive use of dynamic and purposeful micro-interactions, smooth page transitions, subtle hover effects, animated XP counters, and progress bars. The application will leverage animation to enhance gamification, celebrate achievements, and provide clear feedback during user interactions and theme switching.
